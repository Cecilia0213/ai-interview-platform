# AI Interview Preparation Platform

An AI-assisted interview preparation platform designed to help job seekers improve interview performance through personalized mock interviews, AI-generated feedback, and role-specific practice sessions.

---

## Project Status

🚧 Currently under development as part of my MSc Software Development project at the University of Glasgow.

Expected completion: 2026

---

## Project Motivation

Preparing for technical and behavioural interviews can be challenging due to limited access to realistic practice environments and personalized feedback.

This project aims to provide an AI-powered platform that simulates interview scenarios and delivers actionable feedback to help candidates improve confidence and performance.

---

## Planned Features

### Mock Interview Sessions

- Technical interview simulation
- Behavioural interview simulation
- Role-specific question sets

### AI Feedback Generation

- Answer quality assessment
- Communication feedback
- Improvement suggestions

### Personalized Learning

- Strength and weakness analysis
- Progress tracking
- Adaptive question recommendation

### Job-Specific Preparation

- Software Engineering interviews
- Data Science interviews
- AI / Machine Learning interviews
